# Roguelike_game
Написанная на ванильном JS игра-рогалик со случайной генерацией уровней.

Какие функции реализованы:

●	Генерация карты
●	Генерация комнат, их количества и размер каждой
●	Генерация коридоров
●	Размещение на поле мечей (увеличивают урон) и зелий (восстанавливают здоровье). Работает в том числе на врагов (усиляет их и восстанавливает их здоровье).
●	Перемещение по карте и атака противников
●	Противники также перемещаются и атакуют в ответ, если игрок подходит к ним близко
